## Requirements
- **Python 3.9**(or higher version)
- **NanumBarunGothic Font**(Not necessary) - Can be downloaded from https://hangeul.naver.com/font
- **Required Python libraries** - `gTTS`, `playsound`, `PyQt5`
```
> pip3 install gTTS
> pip3 install playsound
> pip3 install PyQt5
```
